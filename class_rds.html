Detailed Description

A complex sensor which gives the relative position of others robots.

This sensor can possibly return the position of all robots in a 2-meter radius. The protocol is [nb of robots] [x_r1] [y_r1] [x_r2] [y_r2] ... For example : 2 120 65 -20 30 means 2 robots at (120, 65) and (-20, 30) The values returned depend on the choosen mode (cartesian or polar).
