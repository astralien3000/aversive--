Detailed Description

Abstract two-way stream with formatted read and write.

Member Enumeration Documentation

enum FormattedStream::SeparatorMode

Represents the different ways a stream can read string.

Word separators are space, \t and line separators.

Line separators are \n, \r and \0.

Default is word.

Note
If the stream is in binary mode, whatever the string separator might be, a string reading will completely fulfil the user's buffer.
Enumerator
WORD 

Read string word by word.

LINE 

Read string line by line.
