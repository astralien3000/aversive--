Detailed Description

template<buffer_t _SIZE = 0, typename _ElementType = u8>
class Buffer< _SIZE, _ElementType >

Buffer implementation which is thread-safe if there are only one provider and one consumer.

Warning
_SIZE must be a power of 2, it is verified with an assert.
Parameters
_SIZE: the number of elements the buffer can handle at any given time.
_ElementType: type of the buffered elements.
