Detailed Description

template<queue_t _SIZE = 0, typename _ElementType = u8>
class Queue< _SIZE, _ElementType >

Class representing a queue (fifo).

Parameters
_SIZE: the maximum number of elements the queue can handle at any given time.
_ElementType: the type of the elements within the queue.
