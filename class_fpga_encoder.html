Detailed Description

template<typename T, int ADDR>
class FpgaEncoder< T, ADDR >

An incremental encoder, used by Eirbot in 2014.

Parameters
Tthe FPGA register type
ADDRthe FPGA register

This sensor is actually connected to a FPGA, which is viewed as an external memory by the microcontroller. The address is set by the user, so I hope you know how your FPGA works ;) !
