Detailed Description

template<typename T, int ADDR>
class FpgaMotor< T, ADDR >

A propulsion motor, used by Eirbot in 2014.

Parameters
ADDR: the address where to set the motor pwm

This actuator is actually connected to a FPGA, which is viewed as an external memory by the microcontroller. The address is set by the user, so I hope you know how your FPGA works ;) !
