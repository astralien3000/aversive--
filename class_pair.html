Detailed Description

template<typename _LeftType, typename _RightType>
class Pair< _LeftType, _RightType >

Two-member tuple.

Parameters
_LeftType: left member type.
_RightType: right member type.
Note
This class is specialized for two-boolean tuples. See Pair<bool, bool> for more information.
