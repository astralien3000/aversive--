Detailed Description

template<array_t _SIZE = 0, typename _ElementType = u8>
class Array< _SIZE, _ElementType >

Class representing a native array with operator overloading.

Parameters
_SIZE: the amount of elements in the array.
_ElementType: the type of the elements contained within the array.
Note
This class is specialized for boolean arrays. See Array<_SIZE, bool> for more information.
